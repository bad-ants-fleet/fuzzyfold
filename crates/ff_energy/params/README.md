# fuzzyfold's parameter files

Note, the parameter files are taken from the ViennaRNA package
and used to test consistency with ViennaRNA predictions.

Some files have minor modifications compared to the ViennaRNA source files:
 - rna_turner2004 contains an additional Misc parameter.
 - "# END" statements were removed or replaced by "#END" statements.
 - rna_turner2004_ext is the combination of parameters for:
  -- rna_turner2004.
  -- additional special hairpin energies.
